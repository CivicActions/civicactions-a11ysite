# Tools

Essential accessibility tools for testing, development, and design workflows.

## Testing Tools

What we look for: Tools that integrate well with development workflows, provide actionable feedback, and cover both automated detection and manual testing support.

### Top Recommendations
1. **axe DevTools** - Browser extension for comprehensive accessibility testing
2. **WAVE** - Visual feedback tool for identifying accessibility issues
3. **Lighthouse** - Built into Chrome, includes accessibility audits

### Additional Options
- Pa11y - Command line accessibility testing
- Axe-core - JavaScript library for automated testing
- Colour Contrast Analyser - Desktop app for color testing

## Development Tools

What we look for: Tools that help developers build accessible code from the start, with clear documentation and integration options.

### Top Recommendations
1. **ESLint Plugin JSX A11y** - Catches accessibility issues in React code
2. **Axe Linter** - IDE integration for real-time accessibility feedback
3. **Storybook A11y Addon** - Test components in isolation

### Additional Options
- Accessibility Insights for Web - Microsoft's testing suite
- ANDI - Bookmarklet for detailed accessibility analysis

## Design Tools

What we look for: Tools that help designers create inclusive designs with proper color contrast, typography, and layout considerations.

### Top Recommendations
1. **Figma A11y Plugin** - Accessibility checking within design workflows
2. **Stark** - Color blindness simulation and contrast checking
3. **Colour Oracle** - Color blindness simulator

### Additional Options
- WebAIM Contrast Checker - Online contrast verification
- Toptal Color Blind Filter - Browser-based color blindness simulation

## Content Tools

What we look for: Tools that help content creators write accessible, clear content that works well with assistive technologies.

### Top Recommendations
1. **Hemingway Editor** - Readability and plain language checking
2. **JAWS Screen Reader** - Industry standard screen reader for testing
3. **VoiceOver** - Built into macOS, great for basic screen reader testing

### Additional Options
- NVDA - Free, open-source Windows screen reader
- Dragon NaturallySpeaking - Voice recognition software testing
